Kesimpulan:
- Tag style dalam HTML adalah alat yang berfungsi untuk menambahkan sentuhan estetika pada elemen-elemen di halaman web. Dengan kata lain, tag ini dipakai untuk memberikan style atau tampilan pada elemen HTML.
- Atribut media menentukan media atau perangkat apa yang dioptimalkan untuk dokumen tertaut. Atribut ini digunakan untuk menentukan bahwa URL target dirancang untuk perangkat khusus (seperti iPhone), media ucapan, atau media cetak.
- List atau yang kita kenal dengan daftar berurutan merupakan fungsi dalam HTML yang digunakan untuk menampilkan data secara berurutan ke bawah. Dalam HTML penulisan list menggunakan tag 'li'.
